How to Use Bill Calculator

Quick Start

  • Enter Data: Click on Rate or Quantity fields to enter values
  • Auto Calculate: Total is automatically calculated (Rate × Quantity)
  • Serial Numbers: Generated automatically for each row
  • Grand Total: Shows sum of all totals across all pages

Navigation

  • Next Page: When rows are filled, click Next to add more items
  • Previous Page: Navigate back to view/edit previous pages
  • Serial Numbers: Continue sequentially across pages

Settings

  • Rows Per Page: Configure 5-30 rows per page (default: 15)
  • Settings Icon: Click gear icon in top-right corner

Actions

  • Clear Page: Remove all data from current page
  • Clear All: Remove all data from all pages
  • Download CSV: Export all data to spreadsheet file

Keyboard Shortcuts

  • Tab: Move to next field
  • Enter: Move to next row
